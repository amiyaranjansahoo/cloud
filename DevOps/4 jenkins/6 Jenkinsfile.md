## What is Jenkinsfile?
```sh
1.	It's a file that contains pipeline code
2.	We typically keep this in git/any source code repository
3.	We can put pipeline code inside job configuration or we can pull this from git.
4.	Now we have kept the code in side job configuration.
5.  But this is recommended to keep the code in git repo – file name – Jenkinsfile
5.	Let’s save the above code in Jenkinsfile.
```
