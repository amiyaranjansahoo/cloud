## Jenkins Plugins 
```sh
Every feature in Jenkins is a plugin, By installing a plugin we get new features into Jenkins.
```
## Few popular used plugins you use in Jenkins? 
```sh
    ● Github
            Integrates Jenkins with GitHub repositories, enabling webhook-based triggers and status updates for builds.
            It helps automate CI/CD workflows directly from GitHub commits and pull requests.
    ● Maven
            Allows Jenkins to build projects using Apache Maven, managing dependencies and lifecycle phases.
            Ideal for Java-based projects that rely on Maven for compilation and packaging. 
    ● sshagent
            Provides SSH credentials to Jenkins jobs, enabling secure access to remote servers.
            Useful for deployments or pulling code from private repos.
    ● email
            Sends email alerts when a Jenkins job succeeds, fails, or meets specific conditions.
            Provides advanced configuration compared to the default Jenkins email notifier.
    ● Sonarqube
            Connects Jenkins to SonarQube for code quality analysis, sending build results for static code checks.
            Helps enforce coding standards and detect vulnerabilities early in the pipeline.
    ● Nexus
            Integrates Jenkins with Nexus Repository Manager to upload or download artifacts.
            Commonly used for managing build outputs like JARs, WARs, or Docker images in a central repository.
```
